# Student-Performance-Analysis

## Overview
This project analyzes student performance using data analytics techniques and presents insights through an interactive dashboard. The objective is to understand how academic, behavioral, and environmental factors influence exam scores.

## Dataset
The dataset contains information on student characteristics, including:

Academic factors: Hours Studied, Attendance, Previous Scores, Tutoring Sessions
Lifestyle factors: Sleep Hours, Physical Activity, Motivation Level
Environmental factors: Parental Involvement, Teacher Quality, School Type, Family Income, Internet Access, Peer Influence, Distance from Home
Target variable: Exam Score

Derived features such as study levels, attendance levels, and sleep levels were created using binning for analysis.

## Methodology
Data cleaning and preprocessing were performed
Missing values were handled using mode imputation
Exploratory Data Analysis (EDA) was conducted using Python
Key variables were grouped into categories for better interpretation
Insights were visualized using an interactive Power BI dashboard
Dashboard

## The dashboard is divided into two sections:

## 1 - Student Overview and Distribution
Displays distributions of study hours, sleep hours, tutoring sessions, and exam scores
Includes categorical breakdowns such as motivation level, internet access, peer influence, and parental involvement
Provides key summary metrics such as average exam score, attendance, and study hours
## 2 - Performance Insights and Factors
Identifies key drivers of exam performance using a decomposition tree
Compares performance across study levels, sleep levels, and attendance levels
Analyzes the impact of teacher quality and school type

## Key Insights
Study hours are the most significant factor influencing exam performance
Attendance strongly impacts academic outcomes
Sleep has an optimal range, with extremes reducing performance
Environmental factors contribute to variation in student results

## Tools and Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
Jupyter Notebook
Microsoft Power BI

## Conclusion
The analysis demonstrates that student performance is influenced by a combination of effort, consistency, and external factors. The dashboard provides an effective way to explore these relationships and derive meaningful insights.
